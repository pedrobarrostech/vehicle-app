# vehicle-app

Vehicle CRUD Application - Spring + Angular 4

Installing API
-------
```
cd user-rest
gradle build
```

Running API
-------
```
java -jar build/libs/user-rest.jar
```
Open http://localhost:8080


Installing UI
-------
```
cd user-webui
npm install
```


Running UI
-------
```
npm start
```
Open http://localhost:4200
